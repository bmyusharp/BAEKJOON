# 2810. 컵홀더

[문제 보기](https://www.acmicpc.net/problem/2810)

- 풀이

컵홀더의 갯수를 구한 뒤, 컵홀더를 쓸 수 있는 사람 수를 구하면 될 것이다.

양 좌석 끝에는 컵홀더가 있고, 좌석 사이마다 컵홀더가 하나씩 있다고 가정하면 컵홀더는 ((좌석 수) + 1)개가 된다.

커플석의 사이에는 컵홀더가 없고, 커플석은 항상 'LL'로 입력된다. 단, 커플석 두 쌍 사이에는 컵홀더가 있다.

L자리가 두 개 나올 때마다 컵홀더 한 개씩을 빼면 된다. 하지만 여기에선 계산을 간단히 하기 위해 L이 하나 등장할 때마다 0.5개씩 뺀 뒤에 정수형 변환을 해주기로 했다.

```python
N = int(input()) #좌석 수 입력
cupholder = N + 1 #컵홀더 기본값
people = input() #커플석/일반석 입력
for i in people:
    if i == 'L':
        cupholder -= 0.5
cupholder = int(cupholder) #정수형 변환
```

컵홀더를 이용할 수 있는 사람 수를 구하는 것이 문제의 정답이다. 커플석 없이 일반석만 있을 경우, 컵홀더의 수가 이용할 수 있는 사람 수보다 한 개 많기 때문에 이를 제외하고 결과값을 출력한다.

```python
if cupholder > N: #전체 이용자보다 컵홀더가 많은 경우
    cupholder -= 1
print(cupholder)
```
